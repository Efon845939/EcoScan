# **App Name**: EcoScan Rewards

## Core Features:

- Product Scanning: Use the device's camera to scan product packaging.
- Material Identification: Identify the material of the scanned product (glass, plastic, metal, etc.). A tool may be used for cases when AI determines information in the product description would increase confidence of correct recycling category.
- Nearest Recycling Location: Find and display the nearest recycling bin locations for the identified material using geolocation services.
- Recycling Bin Barcode Scanning: Scan a barcode at the recycling bin to verify disposal.
- Points Accrual: Award points to the user upon successful recycling bin barcode scan.
- Points Redemption: Allow users to redeem accrued points for discounts at partner stores.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to represent environmental consciousness and sustainability.
- Background color: Very light desaturated green (#F1F8F1) to create a calming and eco-friendly atmosphere.
- Accent color: Blue (#2196F3) to highlight calls to action and important information.
- Font pairing: 'Poppins' (sans-serif) for headlines, 'PT Sans' (sans-serif) for body text.
- Use clear, intuitive icons to represent different material types and recycling actions.
- A clean, card-based layout that emphasizes key product information. Highlighting photography of recycling in action.
- Subtle animations to provide feedback during scanning and navigation. For example, a rewarding animation upon scanning the recycling bin barcode.
